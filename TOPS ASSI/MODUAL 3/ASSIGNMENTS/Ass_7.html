Q-7 What is the difference between the id attribute and the class attribute of HTML elements? With Example.

1:- id :- Id means identifier we can change specific Html elements.
For Ex:-
<p id="paragraph-1">
This is a paragraph.
</p>

1:- class :- Class attribute is define class in HTML elements,class is one group we can change one specific class elements.
For Ex:-
<p class="p1">
This is second paragraph.
</p>
