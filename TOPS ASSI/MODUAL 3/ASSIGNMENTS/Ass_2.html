Q-2 What are tags and attributes in HTML?

HTML TAGS:-

HTML tags means it's tell the browser how to show content. <HTML> </HTML>

There Are two types of tags:

1.PAIREDTAG:-
FOR Ex., p, title, div, script
start tag: -> <title>
End tag: -> </title>

2.Non Pair / Void / self / empty Tag:-
FOR Ex:-br, hr, link, meta

ATTRIBUTES:-

attributes define property of elements. attributes define additional information.

FOR Ex:-<body bgcolor="red"></body>

bgcolor="red" is define attribute of body.
