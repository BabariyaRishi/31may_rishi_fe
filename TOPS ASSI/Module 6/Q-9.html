Q-9 How can the style/class of an element be changed? in JavaScript

In JavaScript,style/class of an HTML elements using the 'style' property or 'ClasssList' property. Example:-
<script>
Using style:-
var element = document.getElementById('myElements');
element.style.color = 'red';
Using classList:-
var element = document.getElementById('myElement');
element.classList.add('newClass');
</script>
