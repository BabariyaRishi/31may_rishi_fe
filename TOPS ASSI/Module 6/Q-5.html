Q-5 What are undeclared and undefined variables?

1:- undeclared Variable:-
=>An undeclared variable is a variable that has not been declared in the code.
=>It is a variable that has not been declared using the var, let, or const keywords.
Example:-
<script>
console.log(myVariable);
</script>
undefined Variable:-
=>An undefined variable is a variable that has been declared but not assigned a value.
=>It is a variable that has been declared using the var, let, or const keywords.
Example:-
<script>
var myVariable = 10;
console.log(myVariable + 5);
</script>
