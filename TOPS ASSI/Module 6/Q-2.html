Q-2 What is the use of isNaN function?

=>The 'isNaN' stands for NOT A NUMBER.
=>The 'is NaN' function use to check value number or not.
=>'isNaN' function give output in boolean value True/False.
Example:-
<script>
console.log(isNaN(123));
console.log(isNaN('Hello'));
console.log(isNaN(NaN));
console.log(isNaN('123'));
console.log(isNaN(undefined));
</script>
