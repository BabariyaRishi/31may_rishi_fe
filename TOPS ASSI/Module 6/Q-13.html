Q-13 What is the function of the delete operator?

In JavaScript, The delete operator is used to remove a property from a object.
=>Syntax:-
delete objectName.propertyName;
=>Example:-
<script>
let myObject = {
name: 'Rishi',
age: 20
};
console.log(myObject);
delete myObject.age;
console.log(myObject);
</script>
