Q-2 Semantic element in HTML5?

=> The semantic elements are a group of HTML tags.
=> There are few semantic tags:-
Example:-

1:article:-Represents an article or entry posted on the weblog (e.g., blog posts).

2:section:-Defines sections within the document.

3:nav:-Specifies a set of navigation links.

4:aside:-Specifies content aside from the main content of a webpage.

5:header:-Provides header information about a section of a page.

6:footer:-Specifies footer information about a section of a page.
