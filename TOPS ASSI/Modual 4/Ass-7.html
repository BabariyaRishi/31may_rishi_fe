Q-7 How can image repetition of the backup be controlled?

Ans:- Controlling image repetition in HTML can be achieved using CSS.

->This property allows you to specify whether and how a background image should repeat within its containing element.

->There are several values you can use for the background-repeat property:

->There are few properties given below:-

1.repeat (default):-The background image repeats both horizontally and vertically.

2.repeat-x:-The background image repeats only horizontally.

3.repeat-y:-The background image repeats only vertically.

4.no-repeat:-The background image does not repeat.
