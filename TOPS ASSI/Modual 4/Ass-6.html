Q-6 How can the background color of an element be changed?

Ans:- You can change the background color of an HTML element using CSS (Cascading Style Sheets).

->To change the background color of an element, you can use the background-color property in your CSS rules.

->This property can be set to a color name or hex code.

EX:-

<div class = "my-element">
This is my element
</div> 

.my-element{
background-color: red;
color: aliceblue;
font-size: 50px;
}
This is my element
