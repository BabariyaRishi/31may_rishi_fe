Q-4 Name a few CSS style components

Ans:- CSS style component describe the presentation and formatting of a HTML documents.
Ex:-
1->COLOR:-CSS allows you to set the color of text and backgrounds using properties like color and background-color.
p{
color: blue;
background-color: lightgray;
}
2->Font:-You can control the font family, size, weight, and style with properties like font-family, font-size, font-weight, and font-style.
h1{
font-family: Arial, sans-serif;
font-size: 24px;
font-weight: bold;
font-style: italic;
}
3->Spacing and Margins:-Properties like margin and padding control the spacing around elements.
div{
margin: 10px;
padding: 20px;
}
4->Borders:-You can add borders to elements using properties like border, border-width, border-color, and border-radius.
button{
border: 2px solid red;
border-radius: 5px;
}
5->Text Styling:-You can style text with properties like text-align, text-decoration, and line-height.
p{
text-align: center;
text-decoration: underline;
line-height: 1.5;
}
