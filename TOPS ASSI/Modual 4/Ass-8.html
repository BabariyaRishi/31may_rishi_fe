Q-8 What is the use of the background-position property?

Ans:-

The background-position property in HTML and CSS is used to specify the starting position of a background image within its containing element.

->The positioning properties allow you to place an image in a specific location within its container by specifying values for top, bottom, left and right.

->The background position property specifies where an element's background image should be placed relative to its box.

->The background position property specifies where a background image should be placed within its container.

->There are Two Main background position

1.->Relative

2.->Absolute

=>Keyword Values:-
left, center, and right for horizontal position.
top, center, and bottom for vertical position.

=>Length Values:-
You can use specific length units like pixels (px), percentages (%), or any other valid length unit to define the position. For example, 10px 20px specifies a horizontal position of 10 pixels from the left and a vertical position of 20 pixels from the top.

=>Percentage Values: You can use percentages to specify the position relative to the width or height of the containing element. For example, 50% 25% means the center horizontally and 25% from the top vertically.
