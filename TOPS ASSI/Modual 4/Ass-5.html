Q-5 What do you understand by CSS opacity?

Ans:- CSS opacity is a property that controls the transparency or level of transparency of an element in a web page.

->The opacity property takes a value between 0 And 1.

-> where 0 is completely transparent and 1 is completely opaque.

-> A value of 0.5 would make the element half transparent, and a value of 0.25 would make the element a quarter transparent.

Ex:-

<div class="opaque-box">
<p>This is a partially transparent box.</p>
</div> 
.opaque-box p{
width: 200px;
height: 200px;
background-color: red;
opacity: 0.5;
}

This is a partially transparent box.
