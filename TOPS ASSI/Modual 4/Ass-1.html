Q-1 What are the benefits of using CSS?

Css stands for Cascading style sheet is used for describing the presentation of a HTML documents
->It can be applied to both internal as well external css files.
->It helps in controlling the layout and styling of web pages
->There are few benefits to using Css
1->Improved website:- Css can be used to control the appearence of a website,including the fonts,color and layout.
2->Increased accessibility:- CSS can be used to make a website more accessible to users with disabilities. For example, CSS can be used to create larger text, add alt tags to images, and create keyboard-only navigation.
3->Improved maintainability:-CSS can help to make a website more maintainable by separating the styling information from the HTML code. This can make it easier to make changes to the appearance of a website without having to change the HTML code.
Overall, CSS is a powerful tool that can be used to improve the appearance, performance, and accessibility of a website.
