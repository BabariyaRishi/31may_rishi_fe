Q-3 What is the difference between CSS2 and CSS3?

=>CSS3 represents the latest iteration of CSS, offering enhanced and more advanced features compared to CSS2.
=>Let's explore the differences between CSS2 and CSS3 based on various criteria.
[1]Selector Type:-
=>CSS2 employs selectors like element names, IDs, and classes.
=>CSS3 introduces Pseudo-classes, Pseudo-elements, and descendant selectors for more refined selections.
[2]Text:-
=>CSS2 utilizes properties such as font style, font weight, font size, text align, etc., for text formatting.
=>CSS3 expands text styling with features like text shadow, text-overflow, and gradient colors for fonts.
[3]Background Gradient:-
=>CSS2 is limited to solid color backgrounds.
=>CSS3 introduces advanced options like linear-gradient and radial-gradient for creating gradient backgrounds.
[4]Element Position:-
=>CSS2 relies on margin and float for positioning elements on a webpage.
=>CSS3 introduces modern layout techniques like Flexbox and Grid for more sophisticated page layouts.
[5]Animation:-
=>CSS2 lacks facilities for creating animations.
=>CSS3 empowers developers to craft diverse and aesthetically pleasing animations.
[6]Media Queries:-
=>CSS2 struggles to create responsive webpages.
=>CSS3 enables the creation of responsive designs through the use of media queries.
