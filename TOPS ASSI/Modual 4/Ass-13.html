Q-13 What are the ways to integrate CSS as a web page?


Ans:-
=>>There are few ways to integrate CSS (Cascading Style Sheets) into a web page. CSS is used to control the presentation and layout of HTML content on a web page.

1.Inline CSS:-
=>Inline CSS is applied directly within an HTML element using the style attribute. This method is useful for making quick styling changes to specific elements.

Ex:-

<h3 style="color: white; font-size: 30px; background-color: black;">
This is a blue text with a font size of 30px.
</h3>

This is a font size of 30px.



2.Internal CSS (Embedded CSS):-
=>Internal CSS is defined within the HTML document itself, typically in the <style> element within the document's <head> section. It applies styles to the entire page or specific elements using CSS selectors.

Ex:-
<head>
<style>
.Internal{
font-size: 30px;
background-color: aqua;
}
h4{
color: red;
font-size: 35px;
}
</style>
</head>

This is a font size of 35px.



3.External CSS:-
=>>External CSS is the most common and recommended method for applying styles to web pages. Styles are defined in a separate .css file and then linked to the HTML document using the <link> element within the <head> section.
<html>
<head>
<link rel="stylesheet" type="text/css" href="13.css">
</head>
<body>

This is a font size of 32px.

<h5>This is a font size of 32px.</h5>
</body>

*{
margin: 0;
padding: 0;
}
h5{
font-size: 32px;
