Q-15 What are the external style sheets?

Ans:-
=>>External Style Sheet is a file that contains CSS code and it can be used by many HTML files.

=>>External style sheets are typically saved with a .css file extension, indicating that they contain Cascading Style Sheets (CSS) rules.

=>> To apply an external style sheet to an HTML document, you use the <link> element in the HTML's <head> section. Here's an example: Ex:-
<html>
<head>
<link rel="stylesheet" type="text/css" href="13.css">
</head>
<body>

<h5>This is a font size of 32px.</h5>
</body>
<html>

OutPut:-

This is a font size of 32px.

*{
margin: 0;
padding: 0;
}

font-size: 32px;
background-color: brown;
}
