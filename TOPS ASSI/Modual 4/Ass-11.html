Q-11 How to center block elements using CSS1?

Ans:-
->In CSS1, which was the first version of CSS, there were limited capabilities compared to modern CSS. However, you can center block-level elements horizontally using CSS1 by using the text-align property and the margin property. Here's an example:

Ex:-
<div class="centered-box">
<h3>This is a centered block element.</h3>
</div>

.centered-box{
text-align: center;
margin-left: auto;
margin-right: auto;
width: 50%;
background-color: aquamarine;
padding: 20px;
}

This is a centered block element.
