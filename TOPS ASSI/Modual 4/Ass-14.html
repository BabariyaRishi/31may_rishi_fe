Q-14 What is embedded style sheets?

Answer:
Internal Style sheet known as embedded style sheet.

=>>When single HTML document needs to be styled uniquely,internal or embedded css might be employed.

=>>Internal CSS can be used to define styles for a single HTML page.

Ex:-

This is a heading

This is a paragraph with embedded styles.

Code:-

<h2>This is a heading</h2>
<h3>This is a paragraph with embedded styles.</h3>
h2{
font-size: 32px;
}
h3{
font-size: 32px;
}
