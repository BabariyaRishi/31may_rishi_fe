Q-16 What are the advantages and disadvantages of using external style sheets?

Ans:-
Advantages:-
=>>External CSS files can be shared by multiple HTML pages.

=>>You do not need to change your existing HTML code if you decide to make changes in the design layout (for example: colors or fonts).

=>>CSS code is easier to maintain.

=>>It helps us to avoid repetition of same properties throughout different webpages.

=>>We can change styles for all elements on one page at once without changing any other part of the webpage.

Disadvantages:-
=>>Each external style sheet requires an additional HTTP request to fetch the CSS file from the server. This can impact page load times, especially if there are multiple external style sheets.

=>>We have to link an external stylesheet for every page we want to use its styles on.

=>>The browser has to download all the linked stylesheets before rendering any webpage.
