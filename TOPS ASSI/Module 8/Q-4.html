Q-4 What do you know about the Bootstrap Grid System?

The Bootstrap grid system is a powerful tool that allows developers to create responsive and mobile-first website.

The Bootstrap Grid System is a responsive, mobile-first grid system developed by Twitter as part of the Bootstrap framework. It is a powerful and flexible layout system that allows developers to create responsive and consistent designs across various screen sizes.

features of the Bootstrap Grid System:

Responsive Design: The grid system is designed to be responsive, meaning that it adjusts and adapts to different screen sizes, such as desktops, tablets, and mobile devices.

Container: The grid system is wrapped in a container, which helps to manage the layout and ensure proper alignment. There are two types of containers: .container (fixed-width container) and .container-fluid (full-width container).

Rows: Inside the container, the layout is organized into rows. Rows are horizontal groups of columns that help structure the content.

Columns: Columns are the building blocks of the grid system. They are placed inside rows and are used to define the layout of the content. Bootstrap uses a 12-column grid system by default, and you can specify the width of each column based on a 12-column layout.

Column Classes: Bootstrap provides a set of responsive utility classes to define the width of columns. These classes are named based on the device size (e.g., col-sm, col-md, col-lg) and the number of columns (e.g., col-6 for half-width).
