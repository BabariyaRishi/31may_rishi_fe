Q-6 What is a Button Group, and what is the class for a basic Button Group?

A Button Group in Bootstrap is a component that groups multiple buttons together on a single line, either horizontally or vertically.
The class for a basic Button in Bootstrap is btn-group. You can create a group of buttons using the btn class within the btn-group class. Here is an example:
For Example:-
<div class="btn-group" role="group" aria-label="Basic Example">
<button type="button" class="btn btn-info"> Register</button> <button type="button" class="btn btn-danger"> Submit</button> <button type="button" class="btn btn-warning"> Cancel</button> </div>
The btn-info, btn-danger, and btn-warning classes are used to specify the background color of the buttons.
