Q-10 What is a bootstrap card and how would you create one?

A Bootstrap 5 card is a UI component provided by the Bootstrap framework that allows you to organize and display content in a structured and visually appealing manner. Cards in Bootstrap are versatile and can be used to showcase various types of content, such as images, text, links, and more.
